# AI Study Planner

## Overview
AI Study Planner is a web application designed to help students organise assignments, plan revision and generate realistic study schedules.
The project is currently in the research and planning stage.

## Goals
- Organise assignments
- Track deadlines
- Generate study schedules
- Use AI to improve planning
- Build a production-ready portfolio project
